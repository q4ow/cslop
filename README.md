# CSLOP

### (C) Sloppy Loader of Packages

Sensible, inferred extractions for the modern day

## Usage
```
Usage: cslop [OPTION...] ARCHIVE...
CSLOP -- A smart archive extractor that handles various archive formats

  -d, --directory=DIR        Extract files into DIR
  -f, --overwrite            Force overwrite of existing files
  -n, --no-directory         Do not create a subdirectory for extraction
  -p, --preserve             Preserve original file permissions

 Output control:
  -q, --quiet                Suppress all non-error output
  -v, --verbose              Show detailed progress information

  -?, --help                 Give this help list
      --usage                Give a short usage message
  -V, --version              Print program version

Mandatory or optional arguments to long options are also mandatory or optional
for any corresponding short options.

Archive files can be of any supported type (zip, tar, gz, bz2, xz, 7z, rar).
Multiple archives can be extracted at once.

Report bugs to q4ow@proton.me.
```
